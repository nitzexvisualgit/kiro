# Icon Assets

Replace these with your final 23×23 PNG icons before shipping. CEP uses these in:
- AE's Window menu icon next to "Omni Forge"
- The panel's small visual identity

Required files (referenced in `CSXS/manifest.xml`):
- `icon-normal.png`     (default state)
- `icon-rollover.png`   (mouse hover)
- `icon-disabled.png`   (when AE deactivates the panel)

**Specs:** 23×23 px PNG, transparent background, lime accent (#a3e635) on dark.
The brand mark in the panel header (the ⚡) is text-rendered, so you only need these for the AE menu integration.

Until you provide real icons, AE will show a generic placeholder — the panel still loads fine.
