# Omni Forge — OFFLINE EDITION 🟢

> **You are looking at the OFFLINE version.** No internet, no server, no Cloudflare. Activation just works.

Eight surgical workspaces — Forge, Curveset, Typecast, FX Core, Vault, Kinetic, Studio — that cut hours from every motion-graphics project.

## What's inside

| Workspace | What it does |
|---|---|
| **Home**     | Dashboard + quick navigation |
| **Forge**    | Align (9-point), Anchor (9-point), Organize project, Precomp w/ duration trim, **Un-Precomp**, Property Clipboard (5 slots), **Explode Text Pro**, Number Counter, Layer Creation suite |
| **Curveset** | Interactive cubic-bezier easing editor + 32 presets + Apply Easing across selected keyframes + Copy CSS + Save Preset |
| **Typecast** | Project-wide font scanner, replacer, scope filter (Project/Comp), per-selection mode |
| **FX Core**  | Enable / Disable / Remove effects across project, comp, or selection · third-party effect probe + manual registry (Deep Glow, Saber, etc.) |
| **Vault**    | Save & re-apply expressions, animation presets, effect presets, comps to a user-chosen library folder |
| **Kinetic**  | 18+ in/out/loop animations with custom curves, applied to any selected layer |
| **Studio**   | Author info, social links, Patreon/PayPal support, license manager |

## Install

### Windows
1. Unzip the package somewhere (e.g. `Downloads/omni-forge`)
2. Right-click `installers/windows/install.bat` → **Run as administrator**
3. Restart After Effects
4. **Window → Extensions → Omni Forge**

### macOS
1. Unzip the package
2. Open Terminal in the `omni-forge` folder
3. Run: `bash installers/mac/install.sh`
4. Restart After Effects
5. **Window → Extensions → Omni Forge**

> Detailed install + signing guide: see [`docs/INSTALL.md`](docs/INSTALL.md)
